# Lab laravel Dusk

## Travail à faire

Utiliser laravel dusk pour Tester lab-crud-basic.


## Installation


```shell
    composer require --dev laravel/dusk
```

```shell
    php artisan dusk:install
```

## Création des Tests


```shell
    php artisan dusk:make CreateTaskTest
```
```shell
    php artisan dusk
```
